## 2. Что делает продукт

AI-платформа для поиска грантов, тендеров и инвестиционных программ, проверки соответствия
требованиям и подготовки заявок — для **бизнеса** (МСБ, предприниматели) и **молодёжи**
(студенты — стипендии, конкурсы, обмены, стажировки, хакатоны).

Два самостоятельных продуктовых направления на одной платформе:

- **FundingPro Business** — AI-подбор грантов/тендеров/инвестпрограмм, проверка соответствия,
  AI-генерация заявок (концепт-нота, логфрейм, бюджетный нарратив), CRM-трекер заявок,
  консультант-маркетплейс, success-fee 2–5% с выигранной суммы.
- **Opportunities Lab** — направленный AI-онбординг молодёжи: чеклист из 10 шагов от регистрации
  до подачи первой реальной заявки, с прогресс-картой для наставника и сертификатом по
  завершении. Локализация на каракалпакский язык.

## 4. Платежи — реально интегрированы, не заглушка

Три платёжных провайдера Узбекистана, каждый — с реальным протоколом (не мок):

| Провайдер | Протокол | Sandbox E2E |
|---|---|---|
| Uzum | Merchant REST | `npm run uzum:sandbox` |
| Payme | JSON-RPC 2.0, Basic Auth | `npm run payme:sandbox` |
| Click | SHOP Prepare/Complete, MD5-подпись | `npm run click:sandbox` |

Активация подписки — **только** через подписанный серверный webhook, идемпотентность по
`idempotencyKey`, дедупликация событий, аудит-лог на каждое действие. Подписки корректно
истекают через биллинг-период (не бессрочны). Success-fee (2–5% с выигранного гранта) —
реальный ledger в БД (`successFeeRecords`), а не только строка в прайсинге: создаётся
автоматически при переводе заявки в статус «Получено» с указанием суммы, admin-панель
`/admin/success-fees` для управления статусом (ожидает → выставлен счёт → оплачено).

`PAYMENTS_ENABLED=false` в текущей конфигурации — это осознанный gate (см.
[`PAYMENTS-OVERVIEW.md`](./PAYMENTS-OVERVIEW.md)), включается после прохождения sandbox E2E,
не признак нерабочего кода.

## 5. AI — не обёртка над одним промптом

- Провайдер-абстракция (OpenAI/Anthropic/mock) с редакцией PII **до** отправки во внешний
  AI (email, телефон, ПИНФЛ, паспорт, ФИО) — [`lib/ai-gateway.ts`](../lib/ai-gateway.ts).
- Strict-режим в production: при недоступности AI-провайдера платный запрос получает `503`,
  а не тихую подмену на демо-текст — квота пользователя не тратится.
- AI-экстракция грантов: администратор вставляет текст объявления → AI возвращает
  структурированный черновик (донор, секторы, суммы, дедлайн, требования) → админ проверяет
  и сохраняет. Массовый импорт с дедупликацией по `sourceUrl` и названию.
- Match-score — прозрачный, без искусственной инфляции: 0% для нерелевантного гранта, разбор
  причин оценки показывается пользователю (не «чёрный ящик»).
