# FundingPro — пилот для физических лиц

Companion to [`STARTUP_SURVIVAL_PLAN.md`](./STARTUP_SURVIVAL_PLAN.md). Чеклист для первых 10–20 участников с менторским сопровождением («Мой путь»).

## Pre-launch (оператор)

- [ ] Когорта создана в `/admin/lab` с `firstLessonAt` и ссылкой на первый урок
- [ ] Оплата протестирована на preview (ручной proof, если PSP sandbox ещё не готов)
- [ ] Аккаунты менторов с `platformRole=admin` или отдельной mentor-ролью
- [ ] Шаблон welcome в Telegram / email + ссылка на первый урок
- [ ] UTM: `utm_source=telegram&utm_campaign=individual_pilot` на `/auth`

## SLA онбординга участника

| Шаг | Действие участника | SLA ментора | Поверхность |
|-----|-------------------|-------------|-------------|
| 1 | Регистрация + оплата (если платный трек) | Подтверждение оплаты ≤ 24ч | `/dashboard/lab/checkout` |
| 2 | Личный профиль | — | `/dashboard/lab` |
| 3 | Интересы + статус CV | Подтверждение профиля ≤ 48ч | Форма профиля |
| 4 | Загрузка CV | Проверка CV ≤ 72ч | Документы + задача CV |
| 5 | Выбор 1 реальной возможности | Подтверждение ≤ 48ч | Задача chosen opportunity |
| 6 | Мотивационное письмо | Проверка ≤ 72ч | Документы |
| 7 | Подача заявки | Подтверждение ≤ 48ч | Трекер заявок |
| 8 | Загрузка proof | Одобрение proof ≤ 72ч | APPLICATION_PROOF |
| 9 | Сертификат | Финальное одобрение ≤ 48ч | Admin Lab journey |

**North Star:** первое подтверждённое proof заявки в течение **30 дней** после старта когорты.

## Ритм ментора (еженедельно)

| День | Задача | Бюджет времени |
|------|--------|----------------|
| Пн | Пакетная проверка CV и писем | 2ч / 10 участников |
| Ср | Office hour (Telegram/live) | 1ч |
| Пт | Подтверждение proof и подач | 1.5ч / 10 участников |

Правило масштаба: второй ментор, когда активных участников > **15** или медиана ответа превышает SLA на 24ч.

## Эскалация

| Проблема | Владелец | Действие |
|----------|----------|----------|
| Оплата не отразилась | Ops | Admin payments + ручной enroll |
| Участник застрял > SLA | Lead mentor | ЛС + заметки в admin Lab |
| AI mock / низкое качество | Engineering | Проверить AI provider env на prod |
| Возврат | Ops | [`legal/refunds`](./legal/refunds) |

## Критерии успеха пилота (неделя 4)

- [ ] ≥ 35% оплативших с CV, одобренным ментором
- [ ] ≥ 20% с хотя бы одним proof заявки (pending или approved)
- [ ] Медиана первого ответа ментора ≤ 48ч
- [ ] Ноль сертификатов без одобренного proof
